الشروط قبل الطلب

  • ✅ يُرجى إرسال كل التفاصيل قبل البدء (مثل: الأسماء، الألوان، اللغة، التنسيق، والمحتوى المطلوب).

  • 🛠️ يُسمح بتعديل واحد فقط مجانًا بعد التسليم. (تعديلات كبيرة أو إعادة التصميم = تحسب خدمة جديدة).
  • ❌ لا يُقبل التراجع عن الطلب بعد بدء العمل.
  • 📁 التسليم يكون بصيغة PDF / PNG فقط (حسب نوع التصميم).
  • 💰 الدفع قبل بدء العمل أو حسب الاتفاق.
  • 💬 التواصل يكون عبر تيليجرام أو ديسكورد فقط، يُرجى الرد السريع لتسريع التنفيذ.
